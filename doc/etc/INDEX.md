# 가이드 인덱스

## 목적

- ML 학습용 쌍데이터(`raw.txt -> *_최종.md`)를 일관된 품질로 구축한다.
- 변환 규칙, 단어 통일, 과정별 예외를 분리해 재현성을 높인다.
- 대화에서 확정된 규칙을 누적해 반복 판단을 줄인다.

## 적용 범위

- 본 세트는 `라벨 유지본(*_최종.md)` 기준이다.
- 논라벨 파생본 규칙은 현재 범위에서 제외한다.

## 가이드 맵

- `common-guide.md`: 과목 공통 문서 구조/라벨/포맷 규칙
- `word-guide.md`: 본 과목 전용 단어·표현 통일 규칙
- `subject-guide.md`: 본 과목 전용 작업 순서·예외 처리 규칙
- `meta-guide.md`: 과정/렉처/챕터 및 품질 메타데이터 규칙

## 우선순위 원칙

- 원칙: `서브젝트가이드의 선택 규칙(조건 충족) > 공통가이드 필수 규칙 > 단어가이드`
- 단, 단어가이드는 구조를 바꾸지 않고 표현 정규화만 담당한다.
- 의미 왜곡 위험이 있으면 자동 적용하지 않고 사용자에게 질의한다.

## 출처 매핑

- 문서 포맷/구조 규칙: `변환_가이드.md` 기반
- 데이터 수집/메타 규칙: `ML_DATA_COLLECTION_GUIDE.md` 기반
- 단어 라이브러리 기준 파일: `guideDoc/word-library.yaml`

## 업데이트 정책

- 대화 중 재사용 가능한 내용이 나오면 반영 위치(공통/단어/서브젝트/메타)를 먼저 제안한다.
- 사용자 승인 후 가이드에 반영한다.
- 반영 시 날짜와 근거 요약을 각 파일 변경 이력에 기록한다.

## 규칙 ID 체계

- `C-###`: 공통가이드
- `W-###`: 단어가이드
- `P-###`: 서브젝트가이드
- `M-###`: 메타가이드
