# mdSee 기능 목록

## 파일 열기
- 툴바 버튼 또는 Ctrl+O → 파일 다이얼로그
- 드래그 앤 드롭 (.md 파일)
- 실시간 파일 감시 (fs.watch) → 외부 편집 시 자동 새로고침
- 최근 열린 파일 경로 타이틀바 표시

## 창 컨트롤
- 커스텀 타이틀바 (frame: false)
- 최소화 / 최대화 / 닫기 버튼

## 마크다운 렌더링
- GFM (GitHub Flavored Markdown)
- 코드 블록 문법 하이라이팅 (highlight.js)
- 실시간 렌더링

## 목차 (TOC)
- 헤딩(H1~H4) 자동 추출
- TOC 패널 토글 (btnToc)
- 스크롤 연동 (현재 위치 자동 하이라이트)
- 키보드 탐색:
  - ↑↓: 같은 레벨 이전/다음 항목
  - ←: 부모 레벨(상위 헤딩)로 이동
  - →: 자식 레벨(하위 헤딩)로 이동
  - Enter: 해당 헤딩으로 스크롤

## 편집 기능 (편집모드)
- 우클릭 컨텍스트 메뉴 (TOC 항목 또는 마크다운 본문)
- 공통: 우클릭 메뉴에서 `내용 편집` 1개만 제공
- `내용 편집`은 대상에 따라 자동 분기
  - 일반 텍스트: 소스 줄 기준 본문 편집
  - 표: 선택된 셀 기준 편집
- 줄 편집 시 마크다운 구조 태그(`#`, `-`, `1.` 등)와 라벨(`[해설]` 등)은 편집창에서 분리/고정되고 본문만 편집
- 라벨 우클릭 시 라벨 전용 메뉴에서 라벨 종류 변경 지원 (`[정의]~[전환]`)
- 전역 `고급메뉴` 토글
  - 끔(기본): 실행 가능한 항목만 우클릭 메뉴에 표시
  - 켬: 비활성 항목까지 전체 표시
- 위계 조작:
  - 위계 한 단계 위로 (shiftUp): H2 → H1
  - 위계 한 단계 아래로 (shiftDown): H1 → H2
  - 헤더 → 블릿 (toBullet): # 제목 → - 항목
  - 블릿 → 헤더 (toHeading): - 항목 → # 제목
- 저장: Ctrl+S 또는 툴바 저장 버튼
- 되돌리기: Ctrl+Z (파일당 최대 2단계)
- 미저장 변경 시 파일 열기/드래그 로드/앱 닫기 전에 저장 여부 확인 모달 표시

## 읽기모드 (모드 토글 스위치)

### 블록 선택
- 마우스 드래그 시 블록(헤딩/단락/리스트/표 등) 단위로 선택 자동 확장
- 블럭지정 옵션: 자동 / 수동 (우클릭 메뉴에서 전환)

### 클립보드 복사
- 자동 복사: 블록 선택 시 즉시 클립보드 복사
- HTML 클립 모드:
  - 끔 (기본): 표만 HTML 형식으로 복사, 나머지는 텍스트
  - 켬: 선택된 모든 블록을 HTML 형식으로 복사 (Word/PPT 붙여넣기 지원)

### 라벨
- 헤딩/리스트 앞 라벨 표시 (노란색)
- 라벨 감추기 옵션으로 숨김/표시 전환
- 복사 시 라벨 텍스트 자동 제외

### 읽기모드 우클릭 메뉴 옵션
| 옵션 | 기본값 | 설명 |
|------|--------|------|
| 블럭지정 | 자동 | 선택 자동 블록 확장 여부 |
| HTML 클립 | 끔 | HTML 형식 복사 범위 |
| 라벨 감추기 | 끔 | 라벨 표시 여부 |
| 자동 복사 | 끔 | 선택 시 자동 클립보드 복사 |

## 설정 저장
- 모든 읽기모드 옵션은 앱 종료 후에도 유지
- 저장 위치: `%APPDATA%/mdSee/config.json`

## 자동업데이트
- 패키지 앱 실행 시 자동으로 최신 버전 확인
- 새 버전 발견 시 백그라운드 다운로드 후 재시작 적용 안내
- 업데이트 피드 주소는 환경변수 `MDSEE_UPDATE_URL`로 지정

## 단축키 요약
| 단축키 | 동작 |
|--------|------|
| Ctrl+O | 파일 열기 |
| Ctrl+S | 파일 저장 |
| Ctrl+Z | 되돌리기 |
| ↑↓←→ | TOC 키보드 탐색 |
| Enter | TOC 항목으로 스크롤 |
