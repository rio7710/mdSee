# mdSee

Windows용 마크다운 뷰어 — 실시간 렌더링, 편집, 읽기모드 지원

## 설치

[Releases](https://github.com/rio7710/mdSee/releases) 페이지에서 최신 `.exe` 파일을 다운로드해 설치합니다.

## 주요 기능

- 마크다운 실시간 렌더링 (GFM, 코드 하이라이팅)
- 목차(TOC) 사이드바 및 키보드 탐색
- 편집모드: 우클릭으로 줄/셀 편집, 위계 조작
- 읽기모드: 블록 선택, 클립보드 복사 (HTML 지원)
- 실시간 파일 감시 및 자동 새로고침
- 자동 업데이트

## 단축키

| 단축키 | 동작 |
| --- | --- |
| Ctrl+O | 파일 열기 |
| Ctrl+S | 파일 저장 |
| Ctrl+Z | 되돌리기 |
